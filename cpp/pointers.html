1️⃣ What is a Pointer?

A pointer is a variable that stores the address of another variable.

int x = 10;
int *p = &x;
  • x stores value 10
  • p stores address of x
  • *p gives value → 10

2️⃣ Why Do We Need Pointers?

  • Dynamic memory allocation
  • Efficient parameter passing
  • Data structures (LL, Tree, Graph)
  • Direct memory access

⚠ Without pointers, DSA is impossible.

3️⃣ Pointer Syntax

int a = 5;
int *ptr = &a;

cout << ptr;   // address
cout << *ptr;  // value

* → dereference | & → address-of

4️⃣ Call by Value vs Call by Pointer

void change(int *x) {
    *x = 100;
}

int main() {
    int a = 10;
    change(&a);
    cout << a; // 100
}

5️⃣ What is a Reference?

int a = 10;
int &ref = a;

ref = 20;
cout << a; // 20

✔ No separate memory | ✔ Cannot be NULL | ✔ Cannot be reassigned

6️⃣ Pointer vs Reference
