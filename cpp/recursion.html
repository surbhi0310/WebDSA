1️⃣ What is Recursion?

Recursion is a programming technique where a function calls itself to solve a problem by breaking it into smaller subproblems.

Every recursive solution has two parts:

  • Base Case – stops recursion
  • Recursive Case – function calls itself

2️⃣ Basic Example

void print(int n) {
    if(n == 0) return; // base case
    cout << n << " ";
    print(n - 1);      // recursive call
}

3️⃣ Understanding the Call Stack (VERY IMPORTANT)

Each recursive call is stored in the call stack. When the base case is reached, stack frames start resolving.

  • Too many calls → Stack Overflow
  • Stack stores local variables & return address

4️⃣ Types of Recursion

  • Tail Recursion
  • Head Recursion
  • Tree Recursion
  • Indirect Recursion

5️⃣ Time & Space Complexity in Recursion

Time complexity depends on:

  • Number of recursive calls
  • Work done in each call

Space Complexity = height of recursion tree (stack space)

6️⃣ Common Interview Mistakes

  • Missing base case
  • Wrong base condition
  • Infinite recursion
  • Ignoring stack overflow

7️⃣ Recursion + Backtracking

Backtracking is recursion where we explore all possibilities and undo steps.

  • Subsets
  • Permutations
  • N-Queen
  • Rat in a Maze

8️⃣ Practice Questions (PLACEMENT)

  • Print numbers using recursion
  • Factorial & Fibonacci
  • Check palindrome using recursion
  • Generate all subsets
